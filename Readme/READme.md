# RecruitIQ

AI-Powered Recruitment and Candidate Screening Platform

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT Authentication
- PyMuPDF
- XGBoost
- LangChain
- ChromaDB
- Gemini API
- Redis
- Streamlit

## Features

- User Authentication
- Job Management
- Resume Upload
- Resume Parsing
- Skill Extraction
- Candidate Matching
- AI Candidate Insights
- Interview Question Generation
- RAG HR Assistant
